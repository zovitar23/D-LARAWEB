# Mor Temalı Web Sitesi

Bu basit bir mor temalı web sitesidir. 3 sayfadan oluşur: Ana Sayfa, Hakkında ve İletişim.

## Kullanım

Tarayıcınızda `index.html` dosyasını açın.

## Sayfalar

- `index.html`: Ana sayfa
- `about.html`: Hakkında sayfası
- `contact.html`: İletişim sayfası

## Stiller

`style.css` dosyasında mor tema tanımlanmıştır.
